ease-wave-text
Letter Wave Animation

A pure CSS sequential letter wave utility. Each character animates with a staggered delay — creating a smooth, flowing ripple effect. No JavaScript. No dependencies.

Hero Usage

Hello World

.wave-text hover to pause
<!-- Wrap each letter in a span --> <p class="wave-text"> <span>H</span><span>e</span>... </p>
Speed Variants

Gentle

.wave-text-slow 2.4s

Energy

.wave-text-fast 0.6s

Success

.wave-text-color accent #4fffb0
